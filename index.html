Iniciando con el procesador de css

Un preprocesador CSS es una herramienta que nos permite generar, de manera automática, hojas de estilo, añadiéndoles características que no tiene CSS, y que son propias de los lenguajes de programación, como pueden ser variables, funciones, selectores anidados, herencia, etcétera.


/* Evita la repetición */
/* Tiene como parámetro el color */
/* Pixeles por defecto en caso de no enviar parámetros */
@mixin figuras ($fondo, $ancho:200px) {
    background-color: $fondo;
    width: $ancho;
    height: 30px;
    margin-top: 5px;
}

.figura-1 {
    /* Llama el mixin con includes */
    @include figuras(#fff, 100px);
}

.figura-2 {
    @include figuras(#222);
}

.figura-3 {
    @include figuras(rgb(29, 163, 11), 300px);
}

.figuras {
    h3 {
        color: $thirdColor;
    }
}
